# 課題：生命の自律的描画システム（Turing Pattern）の再現

## 1. 背景：マクロな視点

生物の体表模様は、個別の細胞がどこに色を塗るか知っているのではなく、化学物質の相互作用によって「勝手に浮かび上がる」現象（自己組織化）である。本課題では、この「生命の計算」をシミュレーションする。

この「チューリング・パターン」は、かつて**「数理生物学最大の謎」**と呼ばれ、学会では数十年にわたり激しい論争が繰り広げられたテーマです。

難解なミレニアム懸念から離れ、**「なぜシマウマは縞々なのか？」という、誰もが子供の頃に抱く疑問を解き明かす「生命の設計図」を巡る物語**として課題を再構成します。

---

### 学会での議論：何が問題だったのか？

1. 「神の指先」vs「自律的な計算」:
   
   かつて、生物の模様は「遺伝子がピンポイントで場所を指定して色を塗っている」と考えられていました。しかし、アラン・チューリング（天才数学者）は、「そんな複雑な命令はない。単純な数式（反応と拡散）だけで、勝手に模様は浮かび上がる」と主張しました。

2. 証明まで40年:
   
   チューリングの論文（1952年）はあまりに難解で、当時の生物学者は「数式だけで模様ができるわけがない」と相手にしませんでした。しかし、1995年頃に日本の近藤滋教授らが「熱帯魚の模様が成長に合わせて動くこと」を証明し、世界中に衝撃を与えました。

今回の課題は、この**「生命が自ら模様を計算して描き出す瞬間」**を再現するものです。

## 2. 課題の目的

- **数式解析（Maxima）**: 美しい数式表記を用い、模様が発生するための化学的平衡点 (u^*, v^*) を算出する。
- **連成計算（Python）**: Maximaから受け取った平衡点を「種」として、時間とともに「模様の芽」が育つ過程を可視化する。

## 3. 連携の仕様

- `analysis.mac` (Maxima): 定常解を求め `params.json` を出力。
- `simulation.py` (Python): `params.json` を読み込み、模様の動的変化を計算。

## 4. 学術的意義

このモデルは、単なる模様だけでなく、指の形成、肺の構造、血管の配置など、生命の形作り（形態形成）の基本原理として学会で議論されている。

## 2. 数理モデル：Schnakenberg（シュナケンバーグ）系

生物の体表で2つの物質が「反応」し「拡散」することで模様が生まれる。 

- 定常状態（平衡点）の導出 (Maxima):
  
  $u^* = a+b, \quad v^* = \frac{b}{(a+b)^2}$

- 支配方程式 (Pythonによる数値計算):
  
  $\begin{cases} \frac{\partial u}{\partial t} = a - u + u^2 v + D_u \nabla^2 u \\ \frac{\partial v}{\partial t} = b - u^2 v + D_v \nabla^2 v \end{cases}$
