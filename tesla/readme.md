## 1. 課題の背景

ニコラ・テスラは、2つのコイルを同じ周波数で共振させることにより、空間を隔てて電力を伝送する「電磁共鳴方式」を考案しました。本課題では、この送電側（Primary）と受電側（Secondary）の結合回路モデルを解析し、伝送効率が最大化する条件を導き出します。

## 3. 数式モデル（Maximaへの入力指針）

送電側と受電側の回路方程式は、相互インダクタンス $M$ を用いて以下のように記述されます。

$V_s = I_1(R_1 + j\omega L_1 + \frac{1}{j\omega C_1}) - j\omega M I_2$

$0 = I_2(R_2 + j\omega L_2 + \frac{1}{j\omega C_2} + Z_L) - j\omega M I_1$

ここで、$Z_L$ は負荷抵抗、$M = k\sqrt{L_1 L_2}$ です。

2. **伝送効率** $\eta = \frac{P_{load}}{P_{in}}$ の式。

## 考察：テスラ・コイル送電の理想と実装の壁

本解析によって得られた電力曲線（Tesla Power Curve）は、無線送電における**「効率（パワー）」**と**「安定性（運用性）」**の激しいトレードオフを可視化している。

### 1. グラフの3領域と現実の対応

グラフ上の各点は、ワイヤレス給電における異なる設計思想を象徴している。

- **理論的ピーク (PEAK: 約4762W / 赤点)**
  
  - **物理的状態**: 送電側と受電側のインピーダンスが完全に一致した「臨界結合」の状態。
  
  - **現実**: 理論上の最大効率だが、グラフが針のように鋭いため、わずか数ミリの距離変化（$M$の変動）で電力が激減する。テスラが夢見た「世界システム」が直面した、制御不能な不安定さの象徴である。

- **走行中EV給電イメージ (Dynamic EV / オレンジ点)**
  
  - **物理的状態**: ピーク手前の「急斜面」に位置する。
  
  - **現実**: 車高（数十センチの距離）があるため結合係数が低く、パワー不足に陥りやすい領域。現代の技術では、この斜面を登り切って「ピーク」を維持するために、高速な周波数追従制御が研究されている。

- **スマホ給電イメージ (Smartphone Qi / 紫点)**
  
  - **物理的状態**: ピークを超えた右側の「なだらかな裾野」。
  
  - **現実**: パワーは最大値の20%程度（約900W付近の比率）に落ちるが、距離の変化に対して出力変動が極めて緩やかである。ユーザーがデバイスを多少雑に置いても充電が維持される「実用的な安定性」を優先した領域である。

### 2. なぜテスラは電線に勝てなかったのか

今回の解析結果から、以下の2点が工学的な敗因として結論付けられる。

1. **「鋭すぎるピーク」の制御不能**: コンピュータ制御のない時代、この鋭い5000W級のピークを、移動する受電体（ユーザー）に対して維持し続けることは不可能だった。

2. **インフラとしての信頼性**: 電線（有線）は、接続さえされていれば常に95%以上の高い安定領域を維持できる。一方、テスラの無線方式は「奇跡的な1点（ピーク）」を外すと、たちまち低出力な裾野へと転落してしまう。

### 3. 未来への提言：900Wを5000Wへ変えるために

現代においてテスラの夢を再構築するには、単なる「回路設計」ではなく「動的制御」が不可欠である。

本シミュレーションの $M$（相互インダクタンス）が変化した瞬間に、回路のパラメータ（周波数 $w$ や静電容量 $C$）をリアルタイムで書き換え、**「常にグラフの頂点が自分を追いかけてくる」**ようなソフトウェア制御こそが、無線送電を電線に勝らせる唯一の道である。
